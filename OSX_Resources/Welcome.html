About the OPeNDAP C API Library Package

The OPeNDAP C API (oc) provides a set of C-language type definitions and functions that can be used to retrieve data over the internet from servers that implement the OPeNDAP Data Access Protocol (DAP 2.0). The oc implementation provides a low-overhead way to retrieve this data, making it practical to include OPeNDAP capability in relatively simple software.

Included in the oc distribution is:

  • C-language API library
  • a standalone example application octest

This package is required for other Mac OS/X packages OPeNDAP has written. By default it will use /usr as the base directory for installation.
